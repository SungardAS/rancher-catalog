# rancher-catalog
A catalog of services for Rancher

## Zookeeper

An Apache Zookeeper cluster of 3-5 nodes that uses the offical [image][docker-zookeeper-url] from Docker Hub.

## Convoy EBS

A single contianer service that will enable the [Convoy
EBS][convoy-ebs-url] storage driver for the host that it is deployed on.
